Help: wgTeams

Description

The XOOPS module wgTeams you can use for presentation of one or more teams

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgteams folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Basics:

  • You can define one or more teams
  • You can define one or more members
  • You can create relations between a member and one or more teams
  • You can put additional information to each relation
Styles:
  • Varios table styles
  • Various image styles
  • Varios posibilities for positioning a images
