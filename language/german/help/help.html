Hilfe: wgTeams

Beschreibung

Dieses XOOPS-Modul wgTeams kann zur Präsentation von einem oder mehrerer Teams verwendet werden

(De-)Installation

Keine speziellen Massnahmen erforderlich, bitte führen Sie den Standardinstallationsprozess durch: Entpacken die den Ordner wgteams in das Verzeichnis '../modules'. Installation über Administration -> System Module -> Module.

Detailierte Anweisungen zur Installation von Modulen finden Sie unter XOOPS Operations Manual

Funktionen

Grundsätzliches:

  • Sie können ein oder mehrere Teams definieren
  • Sie können ein oder mehrere Mitglieder definieren
  • Sie können Beziehungen von Mitglieder zu einem oder mehreren Teams definieren
  • Sie können zu jeder Beziehung zusätzliche Informationen angeben
Styles:
  • Verschiedene Tabellen-Styles
  • Verschiedene Bilder-Styles
  • Verschiedene Möglichkeiten zur Bildpositionierung
